Equals sign:

Considering more carefully one finds that the well-known equals sign is used often with different meanings:

If we write we mean "numerically equal", i.e. the numerical values are identical after calculation. In this sense the equals sign is used in the equation : the numerical value of the first coordinate of the point in the shifted coordinate system is equal to the number of the 1-coordinate in the old system. Accordingly,  the numerical value of  the 3-coordinate in the shifted system is smaller by than the corresponding  number in the old coordinate system.

If, however, we write we mean: the point "is in the coordinate system represented by"  the three given coordinate numbers. In this equation it needs to be specified in which system the coordinates have been measured. Usually one takes the point of view, this is signified by the symbols for the coordinates which remind us of the choice of our coordinate axes. Care is also needed if we want to represent the point in the shifted system. On no account can we simply  write, because from this you could conclude whicht is definitively wrong as we have seen.

There are obviously three ways out of this difficulty:

1) Either we define a new sign  for "is represented in the system ... by ... ", while we add a symbol for the coordinate system to the equals sign: possibly " " with the meaning " is represented in the system by ... ", e.g., . But  this is complicated and difficult to realize with computer fonts.

2) Or we use the hat which labels the shifted coordinate system to also label the indices, e.g., to remind us that  the coordinates of the old point are now being given with respect to the new-, - or -axis of the system. This too is very difficult to realize on the computer, and moreover uncommon.

3) Therefore, we choose here the third possibility: We place the hat on the coordinates, thus as done with alternative 1), avoiding, however, the statement: "it is represented in the system by ... " in equations.

 

Now that we have clarified this problem, there is no more reason for insecurity or misunderstanding.
